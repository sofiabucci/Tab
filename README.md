# Tâb Game 

A digital implementation of the ancient Egyptian board game **Tâb**.

## Features
* **Modes:** Local PvP, AI (Easy/Medium/Hard), and Online Multiplayer.
* **Online:** Supports multiple servers with real-time updates.
  * Official server: "http://twserver.alunos.dcc.fc.up.pt:8008"
  * Group 4 server: "http://twserver.alunos.dcc.fc.up.pt:8104"
* **System:** Authentication, Rankinngs, and Stick Dice animations.

## Quick Start
1.  Install dependencies: `npm install`
2.  Start server: `node index.js`
3.  Open browser: `http://localhost:8104`
